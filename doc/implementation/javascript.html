Javascript Implementation

The C++ implementation can be compiled with emscripten to produce a Javascript implementation of Jsonlang. This is how we implement our Jsonlang demo. It is not the fastest possible Javascript implementation of Jsonlang, but it is easy to build, and is sufficient for many purposes.

To compile it, first install emscripten and ensure em++ and emcc are in your path. Then make libjsonlang.js.

An unofficial nodejs package of Jsonlang is also available.
